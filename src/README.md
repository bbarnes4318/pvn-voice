# DebtHalf.com Landing Page

A high-converting click-to-call landing page for debt settlement services built with Next.js and Tailwind CSS.

## Features

- Mobile-first responsive design
- Fast load times and optimized performance
- Multiple call-to-action points
- Trust elements and social proof
- Persuasive copywriting focused on benefits
- Clear and compelling value proposition
- Sticky mobile call button for improved conversion

## Tech Stack

- **Next.js**: React framework for fast, SEO-friendly pages
- **TypeScript**: For type safety and improved developer experience
- **Tailwind CSS**: For rapid UI development
- **React Icons**: For high-quality icons
- **Responsive Design**: Optimized for all device sizes

## Getting Started

### Prerequisites

- Node.js 14.x or higher
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

Build the application for production:

```bash
npm run build
# or
yarn build
```

Start the production server:

```bash
npm run start
# or
yarn start
```
